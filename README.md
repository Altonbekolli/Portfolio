# Alton Bekolli Portfolio

Willkommen in meinem Portfolio Repository.

Auf dieser Webseite stelle ich mich, meinen bisherigen Werdegang und eine Auswahl meiner Projekte vor. Das Portfolio dient als zentrale Übersicht über meine bisherigen Erfahrungen in der Webentwicklung, Softwareentwicklung und im Studium der Informatik an der Technischen Hochschule Ulm.

## Über mich

Ich bin Alton Bekolli und studiere Informatik an der Technischen Hochschule Ulm mit den Schwerpunkten Mobile Development und Computer Engineering.

Seit April 2024 arbeite ich als Werkstudent bei der BITE GmbH. Dort konnte ich praktische Erfahrung in der Frontend und Backend Entwicklung sammeln und eigenständig an realen Projekten mitarbeiten. Mein Praxissemester habe ich dort ebenfalls erfolgreich absolviert.

Ich interessiere mich besonders für moderne Webentwicklung, Backend Systeme, API Entwicklung, Datenbanken sowie technische Themen rund um GPU, Rendering und Softwarearchitektur.

## Verwendete Technologien

Für dieses Portfolio wurden HTML, CSS und JavaScript verwendet.

In meinen weiteren Projekten arbeite ich außerdem mit React, React Router, PHP, Go, C#, Blazor, ASP.NET Core, Fluent UI, PostgreSQL, OpenGL, Rust und weiteren Technologien aus Web und Softwareentwicklung.

## Inhalte der Webseite

Die Webseite enthält eine persönliche Vorstellung, Informationen über meinen bisherigen Werdegang, eine Übersicht über meine technischen Fähigkeiten sowie eine Sammlung ausgewählter Projekte.

Zusätzlich gibt es ein Kontaktformular, über das eine direkte Kontaktaufnahme möglich ist.

## Projekte

Im Portfolio sind verschiedene Projekte aus unterschiedlichen Bereichen eingebunden.

Dazu gehören Webprojekte, Studienarbeiten, praktische Arbeiten aus dem Praxissemester sowie technische Projekte aus der Softwareentwicklung.

Einige Beispiele sind ein Raytracer mit C++, OpenGL und GPU Bezug, Projekte aus dem Praxissemester, eine Seminararbeit zum Thema Memory Safety in Rust ohne Garbage Collection sowie weitere Webanwendungen und Systemprojekte.

## Ziel des Portfolios

Das Ziel dieser Webseite ist es, einen strukturierten und authentischen Überblick über meine Kenntnisse, meine Entwicklung und meine Projekte zu geben.

Das Portfolio wird regelmäßig erweitert und an neue Projekte sowie neue Erfahrungen angepasst.
